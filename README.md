# 🛒 ProStore – E-Commerce Platform

ProStore is a full-stack MERN e-commerce web application that provides a complete online shopping experience with secure authentication, product management, shopping cart functionality, payment integration, and admin controls.

Built using **MongoDB, Express.js, React.js, Node.js, and Redux**, the platform allows users to browse products, add items to cart, place orders, make payments, and track purchases. Administrators can manage products, users, and orders through a dedicated dashboard.

## ✨ Key Features

- User authentication & authorization (JWT)
- Product browsing and search
- Shopping cart & checkout system
- PayPal payment integration
- Product reviews & ratings
- Order management
- Admin dashboard
- Product CRUD operations
- Responsive UI design

## 🛠️ Tech Stack

- Frontend: React, Redux, Bootstrap
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT, bcrypt
- Deployment: Render
